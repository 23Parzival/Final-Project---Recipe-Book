PROJECT TITLE: Final Project - Recipe Book

PURPOSE OF PROJECT: To simulate a virtual recipe book in a program.

VERSION: 1.0

HOW TO START THIS PROJECT: In Bluej, create a Controller object and call its start() method. In command prompt, create a new Controller class, then press 1 and the program will start.
